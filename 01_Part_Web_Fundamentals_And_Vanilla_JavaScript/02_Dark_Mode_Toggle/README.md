## 🎯 Project Overview

This project focuses on building a **responsive navigation system with theme switching capability**.

The selected theme (dark/light) is stored in **localStorage**, ensuring persistence even after page reloads.

Key learning areas:
- State management in React  
- Theme persistence using localStorage  
- Responsive UI development  
- Accessibility best practices  
- Modern frontend architecture
